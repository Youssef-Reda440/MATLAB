# Lab 2 – Electrostatic Forces and Vector Coordinate Transformations

## Overview
Lab 2 focuses on the analysis of electrostatic forces and vector operations using MATLAB. 
The lab includes both practical activities and a detailed report covering force computation, visualization, and coordinate system transformations.

---

## Contents
This lab consists of the following components:

- **Activity 1:**  
  Electrostatic force analysis between three point charges, including 2D and 3D visualization.

- **Activity 2:**  
  Vector addition and subtraction where vectors are expressed in Cartesian and Cylindrical coordinate systems.

- **Report:**  
  A complete report that includes force visualization, coordinate transformations (Cartesian, Cylindrical, and Spherical), result validation, and discussion.

---

## Tools Used
- MATLAB
- Vector algebra
- Coordinate transformation matrices

---

## Key Concepts
- Coulomb’s Law
- Vector superposition
- 2D and 3D visualization
- Cartesian, Cylindrical, and Spherical coordinate systems
- Vector transformation versus point conversion

---

## Notes
- All force and vector calculations were validated using analytical methods.
- Vector scaling was applied only for visualization purposes.
- Coordinate transformations were applied to vectors, not points.

---

## Conclusion
Lab 2 provides practical experience in electrostatic force analysis and vector coordinate transformations.
The obtained results demonstrate consistency between MATLAB simulations and theoretical calculations.
